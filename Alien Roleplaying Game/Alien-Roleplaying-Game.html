Change log
v4.03
(2022-04-05)
Fixed spelling mistake in API roll for Weapon 5.
v4.02
(2022-03-30)
Fixed the handling of secret (whispered) rolls so that it works across tabs and different roll types. Also, it no longer resets when opening the sheet. An upgrade function will ensure setting stays same as before.
Please note an upcoming release will fix the Default Sheet Settings, and it seems like there is no way to prevent the game default settings from overriding the local settings for each sheet. This change will be announced later, and will require you to reset your sheet configuraiton if different from the game default.
v4.01
(2022-03-20)
Mobility and Observability macros now work the same for Character and Hostile tab, same macro should be usable for either. Technically, they both use the attribute and the skill, but for the Hostile the attribute would be kept at the default 0 level.
The Roll template Success and Panic calculations are now fixed from the previous version, it kept showing 0 no matter how many successes or facehuggers one rolled.
The Roll template chat output now states Panic for the facehugger results, to make similar to the API Script output.
Selecting secret rolls when using the API Script now honors the setting for rolls using the Roll template.
v4.00
(2022-03-09)
Added the Hostile tab to use for Xenos and other nasty things that lurk in the darkness of space.
Adjusted the size of the sheet frame to remove white border in dark mode. Space is a very dark place.
v3.03
(2022-02-04)
Fixed armament 3 roll button, changed armament roll name to support API, and changed armor rolls to not use API.
v3.02
(2022-01-19)
Fixed armament roll issue by calculating rangemod at sheet load.
v3.01
(2022-01-10)
Fixed macros not working due to roll names being duplicated.
v3.00
(2021-12-28)
Added configuration tab for sheet level options and customizations, like input fields and background image.
Added support for API rolls using the Alien RPG dice roller script. This requires a Pro account.
Added default sheet settings for all the configuration options.
Added weights and encumbrance calculation as an option. Also added a repeating section at the end of teh gear list, to allow adding 9 items or more.
Added option to calculate Health automatcially, and included a Tough checkbox to include in calculation.
Added indicators for tabs of lists making it easier to see where there is data in the tabbed list.
Thanks Richard M for this!
Added a repeating section at end of Gear list to allow for exapanding it, and made the panel scrollable. Also added an option to only use the repeating section, and hide the static gear rowws.
Added four more talent slots.
Added option for squarer input boxes and option to enable spinner (up and down buttons) for number fields.
v2.01
Fixed roll template, readding the AllProps for use in macros.
Fixed counting of successes also for Stress dice.
v2.0
Added the Ship tab.
Added counting of successes and failures in roll template.
v1.03
(2020-11-21)
Added internationalization to support translation of sheet.
v1.02
(2020-08-15)
Updated some of the layout, adjusted to mirror the layout/tone of the game's character sheet and to make it a bit more understandable after play session feedback.
