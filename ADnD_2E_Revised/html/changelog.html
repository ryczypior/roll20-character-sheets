Announcements

Changes in

  • Psionic Powers: Reordered to appear in the same order as presented in the books
  • Psionic Powers: Changed headings to match the books, i.e. Psychokinesis Sciences -> Psychokinetic Sciences
  • Psionic Powers: Fixed a lot of incorrect mouse over texts
  • Psionic Powers Core: Removed Roll modifier as the roll should never be modified. Update macros for this to take effect
  • Psionic Powers Core: Added Misc. modifier to power checks. Update macros for this to take effect
  • Character Revised Psionic Psychokinesis Devotion: Fixed first static row PSP cost field no longer share values with Telepathy Science, first static row PSP cost field
  • Static Follower Core Psionic Clairsentient Sciences: Fixed first macro not printing to the chat
  • Repeating Follower Core Psionic Psychoportive Sciences: Fixed third macro throwing error to the chat
  • Follower Core Psionic: Changed check roll to crit on the Power Score value instead of 1
  • Follower Psionic: Psionic Nonweapon proficiencies now outputs with a roll template
  • Follower Rogue: Changed the layout to be similar to the character sheet with all skills being visible at the same time
  • Follower Rogue: Added armor type selection and armor bonus/penalties modifiers autofill
  • Follower Rogue: Adding %-sign for roll template
  • Characters: Added missing %-sign for Find/Remove Traps, Move Silently, and Hide in Shadows
  • Saving throws now always fail on a natural one, no matter the modifiers. Sage Advice #216 p. 103
  • Nonweapon proficiencies now always fail on a natural 20. PHB p. 75
  • Nonweapon proficiencies now show how well you made your check
  • Nonweapon proficiencies now highlight a critical success
