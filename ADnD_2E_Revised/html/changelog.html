Announcements

Changes in

  • Changed character saving throws to output with roll template
  • Changed character magic resistance to output with roll template
  • Changed followers saving throws to output with roll template
  • Changed Monster saving throws to output with roll template
  • Changed Monster magic resistance to output with roll template
  • Added the new header colors to all roll templates: red, yellow, blue, dark-blue, green, orange, gray, brown, purple, dark-purple
  • The 2Echeck roll template no longer adds "Check vs." to the {{checkvs}} attribute. You can add it yourself if you still want it
  • The 2Einversecheck roll template no longer adds "Check vs." to the {{checkvs}} attribute. You can add it yourself if you still want it
  • No longer stealthily adds the Poison save bonus from high constitution (19 or more) to the save vs poison, as this modifier is not always applied.
    • Dwarves and halflings have a different modifier, and therefore do not use the poison save modifier
    • You can add the bonus yourself to the modifier field next to Poison save, if applicable
